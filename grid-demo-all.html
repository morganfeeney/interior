Griddle

Griddle is a set of fluid and responsive layout tools for front-end web developers. Built from SASS (SCSS) it is configurable to your needs.

@include griddle(
	$grid-cols: 12,
	$grid-gutter-width: 16px,
	$grid-alignment: left
);
