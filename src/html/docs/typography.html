Scale

The font sizes are based upon a typographic scale, traditionally set by printers, adopted for use on the web. The sizes were originally in point yet have been converted to em and are the equivalent in pixel size to what they would have been when set in point. The base font size is 1rem, the base line-height is also 1. The baseline grid is .5rem for the sake of granularity traversing the baseline. I wanted a small viewport body font size of 16px, whilst 18px on larger viewports. In order to do this all fonts scale up by 112.5% for large viewports.
