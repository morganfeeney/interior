Usage

Layouts are created using a grid structure made from a combination of vertical columns laid upon a visual grid, and a variety of — mostly Flexbox — CSS alignment methods. Grid columns are generated via the .column() mixin.

Required arguments

The only required argument is a simple little number.

$columns
The value is used to calculate how many columns from the grid you want to span.

$calc

$margin
Aligns grid content with non-grid content using a negative margin. Otherwise centers the column.
