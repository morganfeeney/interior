Usage

Layouts can be created using a grid structure, made from columns, whilst allowing Flexbox to take care of alignment. A grid can be constructed in many ways by using the .column() mixin and then passing in parameters as needed.

Required arguments

The only required argument is numeric. The value is used to calculate the number of columns out of the total passed to the $cols variable.

$col
The number of grid columns the column must span.

Optional arguments

To omit one of the following default arguments, pass a false value to the argument when calling the mixin.

$calc
Subtracts the $grid-gutter from the column width using CSS calc().
$margin
Aligns grid content with non-grid content using a negative margin. Otherwise centers the column.
