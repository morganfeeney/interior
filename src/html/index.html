Web layout design is evolving.

Something big is happening, specification is pushing the web forward in an unprecedented way. Those ready to embrace change are on the verge of a new era. However, we mustn't forget that beyond the web is a wealth of layout design resource to learn from.

Web and print are different beasts

Paper will fold yet it doesn't shrink. You needn't worry about a paper layout bursting at the seams when it's orientation changes. The fluidity of web alone presents a unique set of challenges, the world of print won't experience.

The transition/translation from print to web is challenging, applying that 'perfect' layout you created in Illustrator to the browser, without also solving content reflow issues isn't straight forward. Besides... we've been limited by browser capability — but we're moving forward.

We finally have the tools necessary to create amazing page designs on the web

Jen Simmons

The capability of browsers and the desires of designers have been at odds for years — this has lead to unreliable hacks, javascript frameworks for presentation, and float based grid systems for scaffolding. They are workarounds, problematic, and prone to break when pushed to extremes. The desire for the ability to create stunning designs in the browser has always been there, yet until recently we haven't had the same level of precision we have with DTP applications with plain old CSS and HTML.

New CSS layout tools

Interior v.2 is intended for use with Flexbox, according to this article in 2017 we'll have the eagerly awaited Grid.

Time to see what the shiny new toys can do

It's so important that we move on from how things have been. In order to do this we need direction. People often draw a line between web and print, I see the line becoming blurred.

The talk Real Art Direction on the Web by Jen Simmons has been a wake up call, it's reminded me of some fundamentals I learnt back in Art College when I was 17 — I'm now 38 — thanks for that.

I suppose the main thing I'm reminded of is, back before I had access to the internet, going to the library digging out some books and learning about the work of artists & designers from the 20th century. I always enjoyed that part of the process, somehow it's got lost along the way.

the medium of web is coming of age

The web is still so young, whereas print is mature. In her talk Jen focuses on the typical layouts we are so used to seeing on the web every day, 3 column layouts for example, and is 100% right for illuminating how devoid of creativity they are. These layouts have become normalised.

Taking the time to stop and re-assess things is therefore crucial in order to move forward and break free of old habits. Researching the works of layout specialists from the world of print design is defintiely a way forward, however web is it's own medium with it's own unique challenges and features. It needs to evolve into its own thing, yet draw upon the influence that is required to imagine the possibilities.
