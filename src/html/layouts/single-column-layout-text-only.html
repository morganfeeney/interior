But aren't web and print different beasts?

Paper can fold yet it doesn't shrink. You needn't worry about a paper layout bursting at the seams when it is rotated on it's side. The fluidity of web browsers alone presents us with a unique set of challenges those in the world of print don't experience.

Unreliable hacks, dependency upon javascript frameworks, and float based grid systems are problematic, and will eventually break if you push too hard. We now have a range of different features to play with.
