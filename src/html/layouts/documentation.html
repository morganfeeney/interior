Grid

Layouts can be created using a grid structure made from columns. A grid can be constructed in many ways by using the .col() mixin and then passing in parameters as needed.

Required arguments

The only required argument is numerical. The value is used to calculate the number of columns out of the total passed to the $cols variable.

$col
The number of grid columns the column must span.

Optional arguments

To omit one of the following default arguments, pass a false value to the argument when calling the mixin.

$calc
Subtracts the $grid-gutter from the column width using CSS calc().
$margin
Aligns grid content with non-grid content using a negative margin. Otherwise centers the column.

Baseline

Baseline is a vertical grid for aligning type. The baseline() mixin accepts a variety of arguments.

Arguments

There are no required arguments.

$font-size
Sets the font size. I use a range of variables based upon a typographic scale.
$unit
Multiply the scale of line-height x2, x3 etc.
$line-height
Apply line-height, it should be unitless.
$margin
Adds a margin-top for additional vertical alignment where needed.
$above
Increases the padding above the text, works in conjunction with $below.
$below
The amount of margin-bottom to apply.
