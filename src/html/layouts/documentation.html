Grid

Layouts can be created using a grid structure made from columns. A grid can be constructed in many ways by using the .col() mixin and then passing in parameters as needed.

Required arguments

The only required argument is numerical. The value is used to divide the number of columns passed to the $cols variable.

$col
The number of grid columns the column must span.

Optional arguments

To omit one of the following arguments pass it a false value when calling the mixin.

$calc
Subtracts the $grid-gutter from the column width using CSS calc().
$margin
Aligns grid content with non-grid content using a negative margin. Otherwise centers the column.
