To decide upon a suitable range of media to design for, via Google Analytics, I discovered the following sizes to be the most popular:

  1. 320x568
  2. 360x640
  3. 375x667
  4. 414x736
  5. 768x1024
  6. 1366x768

Some dimensions are considered standard, such as an iPad (768x1024), or a standard Android device (360x640).

With the dimensions I then created what I refer to as the Container Grid.

The purpose of the Container Grid is to create a series of columns that equal the total of each chosen width. If a screen is larger than a width, any excess space is fluid and can be utilised to solve a range of design problems fig. 4.
