Column width not only creates a definite structure for laying out documents, it aids legibility. The well-known rule of 7-10 words per line can be used as a metric for deciding upon a suitable line-length for layout, after all, it aims to rid the fatigue reading long copy creates over time. The combination of using non-fluid screen sizes, with columns based on line-length will also add a tremendous level of certainty.
