Why?

The Baseline Grid and Modular Scale go hand-in-hand, for instance; the baseline grid is used to visually align text by indicating the location of the baseline, its proportions are created using the same proportions the text styles are created with. Interior makes full use of the grid by altering it’s proportions using percentages on the HTML element. This way a single rem unit (the measurement used for incrementally traversing the baseline grid) will be relative to the percentage inherited throughout the entire document. It eventually makes life easy.
