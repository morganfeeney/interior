The sizes within the typographic system are set in points, which is a universal unit of measurement. I initially setup a Sketch library containing these sizes, from there I translated them into CSS for web use, e.g. 1rem equals 16pt/16px.

Type sizes in 12 - 48pt, SF Pro Display, as created in Sketch library (displayed at actual size). The lines indicate where the ascenders and descenders should align.

Type Sizes taken from the Sketch library, translated into CSS custom propertie, which are also interchangeable.

The system functions across a range of screen sizes, font sizes are universally available and interchangable as and when required. With CSS this is achieved through the use of custom properties fig. 2.
