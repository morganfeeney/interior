Typographic system

{% endblock %} {% block main %}

Sizes within the typographic system are set in pt (points), which is a universal unit of measurement. I initially setup a Sketch library containing the sizes in fig.1, from there I translated them into CSS. 1rem equals 16pt in Sketch or 16px on the web.

The system functions across a range of screen sizes, font sizes are universally available and interchangable as and when required. With CSS this is achieved through the use of custom properties fig. 2.
